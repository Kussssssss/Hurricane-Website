Hurricane Trajectory Analysis

Interactive visualization and prediction of hurricane patterns

About the Application

The Hurricane Trajectory Analysis application is an interactive dashboard that allows you to explore hurricane trajectory data, visualize patterns, and predict hurricane categories based on trajectory features. Using the HURDAT2 dataset with 1,903 hurricane trajectories across 6 categories, this application provides valuable insights into hurricane behavior and characteristics.

Category 0: Tropical Depression Category 1: Tropical Storm Category 2: Category 1 Hurricane Category 3: Category 2 Hurricane Category 4: Category 3 Hurricane Category 5: Category 4-5 Hurricane

Key Features

Interactive Map Visualization

Explore hurricane trajectories on interactive maps with filtering by category. View 2D maps, 3D visualizations, and animated hurricane paths to understand movement patterns.

Feature Analysis

Analyze key features of hurricane trajectories including velocity, acceleration, trajectory length, and geographical extent. Explore correlations and distributions across different categories.

Prediction Model

Use machine learning to predict hurricane categories based on trajectory features. Explore model performance, feature importance, and make predictions for custom trajectories.

Trajectory Comparison

Compare hurricane trajectories across different categories. Visualize normalized trajectory patterns and identify similarities between hurricanes.

Hurricane Impact

Visualize potential impact areas based on hurricane trajectory and category. Explore risk assessment and historical comparisons with similar hurricanes.

Data Insights

Gain valuable insights into hurricane patterns, behavior, and characteristics through comprehensive data analysis and visualization tools.

Technical Details

The application is built using:

  • Streamlit: For the interactive web interface
  • Pactus: For hurricane trajectory dataset and analysis
  • Plotly: For interactive visualizations
  • Scikit-learn: For machine learning models
  • Pandas & NumPy: For data processing

The machine learning model uses a Random Forest classifier to predict hurricane categories based on trajectory features, with key features including trajectory length, latitude standard deviation, and trajectory duration.
